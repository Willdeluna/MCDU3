# Cockpit Layout System

Cockpit mode organizes the CDU/MCDU, ND, PFD, MCP/FCU, help cards, panel tray, focus overlay, zoom, and task modes.

Protected layouts include:

- FMC focus.
- Navigation.
- Automation.
- Approach.
- Full deck.
- Focused CDU/MCDU, ND, PFD, and MCP/FCU panels.
- Tablet landscape.
- 3456x2234 and Retina-equivalent large-screen layouts.

Visual changes should update the relevant Playwright baselines.
