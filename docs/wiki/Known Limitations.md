# Known Limitations

RFMC is not certified training software and is not approved for real-world aviation operations.

Key limitations:

- Visual screenshots prove render stability, not hardware pixel accuracy.
- PMDG live validation requires Windows + MSFS + PMDG.
- FBW A320 integration remains mock/scaffolded unless separately implemented.
- Airbus is secondary scope.
- Navdata is deterministic trainer data, not global AIRAC data.
- Performance, LNAV, and VNAV are trainer-grade approximations only.

See `docs/KNOWN_LIMITATIONS.md`.
