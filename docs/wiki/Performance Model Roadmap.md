# Performance Model Roadmap

Performance logic is trainer-grade only.

Current and planned use:

- V-speed prediction scaffolding.
- Fuel-to-destination estimates.
- Runway length and insufficient-fuel warnings.
- PERF, TAKEOFF, PROG, FUEL PRED, VNAV, and training guidance integration.

Every public performance description must preserve this disclaimer:

```txt
Trainer-grade approximation only. Not for operational use.
```
