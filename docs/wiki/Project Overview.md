# Project Overview

RFMC is a browser-based avionics procedure trainer for practicing CDU/MCDU flows, route setup, cockpit scanning, ND/PFD interpretation, and trainer-level autoflight concepts.

Current product shape:

- Boeing 737 NG is the primary aircraft family.
- Airbus A320 support is scoped and documented page-by-page.
- Cockpit mode includes CDU/MCDU, ND, PFD, MCP/FCU, focused panels, and task modes.
- Visual baselines protect desktop, tablet, 3456x2234, Retina-equivalent, focused-panel, ND, and PFD states.
- CONTROL mode and MSFS bridge foundations are mock-tested locally; PMDG live validation requires Windows + MSFS + PMDG.

Current status and command evidence live in `docs/STATUS.md`.
