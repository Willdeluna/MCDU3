# Reference Library

This folder tracks visual and procedural reference sources used for measured CDU/MCDU fidelity work.

Do not copy copyrighted source material into the repository unless its license allows redistribution. Prefer metadata records that point to local/private files or public URLs and describe usage restrictions.

## Required Metadata

- `id`
- `source`
- `sourceType`
- `aircraft`
- `variant`
- `unitGeneration`
- `displayType`
- `usageRights`
- `cropRules`
- `measurementUse`
- `notes`
